## 快速开始

### 使用Python读取数据

```python
import json
import pandas as pd

# 读取JSON格式
with open('json/patients.json', 'r', encoding='utf-8') as f:
    patients = json.load(f)

# 读取CSV格式
df_patients = pd.read_csv('csv/patient.csv')
df_visits = pd.read_csv('csv/visit.csv')
df_labs = pd.read_csv('csv/lab_result.csv')
```

### 数据关联查询

```python
# 获取特定患者的所有就诊记录
patient_id = "PATIENT_ID_HERE"
patient_visits = df_visits[df_visits['patient_id'] == patient_id]

# 获取特定就诊的实验室检查
visit_id = "VISIT_ID_HERE"
visit_labs = df_labs[df_labs['visit_id'] == visit_id]
```
