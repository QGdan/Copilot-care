# 数据字典

## 1. 患者基本信息表 (patient.csv)

| 字段名 | 中文名 | 数据类型 | 取值范围 | 说明 |
|:---|:---|:---|:---|:---|
| patient_id | 患者标识符 | 字符串 | 16位哈希值 | 伪标识符，不可逆加密 |
| birth_year_month | 出生年月 | 字符串 | YYYY-MM | 精确到月，保护隐私 |
| gender | 性别编码 | 整数 | 1,2,9 | 1=男, 2=女, 9=未说明 |
| gender_name | 性别名称 | 字符串 | - | 性别中文名称 |
| ethnicity | 民族编码 | 字符串 | GB/T 3304 | 民族标准编码 |
| ethnicity_name | 民族名称 | 字符串 | - | 民族中文名称 |
| occupation | 职业编码 | 字符串 | GB/T 6565 | 职业标准编码 |
| education | 教育程度编码 | 整数 | 1-5 | 1=文盲, 5=大专及以上 |
| education_name | 教育程度名称 | 字符串 | - | 教育程度中文名称 |
| insurance_type | 医保类型编码 | 整数 | 1-6 | 医保类型代码 |
| insurance_name | 医保类型名称 | 字符串 | - | 医保类型中文名称 |
| height_cm | 身高 | 数值 | 100-220 | 单位：厘米 |
| weight_kg | 体重 | 数值 | 30-200 | 单位：千克 |
| BMI | 体重指数 | 数值 | 10-60 | BMI=体重(kg)/身高²(m²) |
| waist_cm | 腰围 | 数值 | 50-150 | 单位：厘米 |
| hip_cm | 臀围 | 数值 | 60-160 | 单位：厘米 |
| smoking_status | 吸烟状态编码 | 整数 | 1,2,3 | 1=从不, 2=已戒, 3=吸烟 |
| smoking_name | 吸烟状态名称 | 字符串 | - | 吸烟状态中文名称 |
| alcohol_status | 饮酒状态编码 | 整数 | 1,2,3 | 1=从不, 2=已戒, 3=饮酒 |
| alcohol_name | 饮酒状态名称 | 字符串 | - | 饮酒状态中文名称 |
| family_history | 家族史 | 字符串 | - | 分号分隔的疾病列表 |
| has_hypertension | 高血压标志 | 布尔 | True/False | 是否患有高血压 |
| has_diabetes | 糖尿病标志 | 布尔 | True/False | 是否患有糖尿病 |
| has_heart_disease | 心脏病标志 | 布尔 | True/False | 是否患有心脏病 |

## 2. 就诊记录表 (visit.csv)

| 字段名 | 中文名 | 数据类型 | 取值范围 | 说明 |
|:---|:---|:---|:---|:---|
| visit_id | 就诊标识符 | 字符串 | 16位哈希值 | 唯一标识一次就诊 |
| patient_id | 患者标识符 | 字符串 | 16位哈希值 | 关联患者表 |
| visit_datetime | 就诊日期时间 | 字符串 | ISO8601 | 格式：YYYY-MM-DD HH:MM:SS |
| visit_type | 就诊类型编码 | 整数 | 1-6 | 1=普通门诊, 2=专家门诊, 3=急诊, 4=急诊留观, 5=普通住院, 6=重症监护 |
| visit_type_name | 就诊类型名称 | 字符串 | - | 就诊类型中文名称 |
| department | 就诊科室 | 字符串 | - | 科室名称 |
| chief_complaint | 主诉 | 字符串 | - | 患者主要症状描述 |
| systolic_bp | 收缩压 | 数值 | 70-250 | 单位：mmHg |
| diastolic_bp | 舒张压 | 数值 | 40-150 | 单位：mmHg |
| heart_rate | 心率 | 数值 | 40-150 | 单位：次/分钟 |
| respiratory_rate | 呼吸频率 | 数值 | 10-40 | 单位：次/分钟 |
| temperature | 体温 | 数值 | 35-42 | 单位：摄氏度 |

## 3. 实验室检查结果表 (lab_result.csv)

| 字段名 | 中文名 | 数据类型 | 参考范围 | 说明 |
|:---|:---|:---|:---|:---|
| lab_id | 检查标识符 | 字符串 | - | 唯一标识一次检查 |

| visit_id | 就诊标识符 | 字符串 | 16位哈希值 | 关联就诊表 |

| postprandial_glucose_mmol_L | 餐后2小时血糖 | 数值 | <7.8 | 单位：mmol/L |
| hba1c_percent | 糖化血红蛋白 | 数值 | <6.0% | 单位：% |
| total_cholesterol_mmol_L | 总胆固醇 | 数值 | <5.2 | 单位：mmol/L |
| ldl_cholesterol_mmol_L | 低密度脂蛋白胆固醇 | 数值 | <3.4 | 单位：mmol/L |
| hdl_cholesterol_mmol_L | 高密度脂蛋白胆固醇 | 数值 | >1.0 | 单位：mmol/L |
| triglycerides_mmol_L | 甘油三酯 | 数值 | <1.7 | 单位：mmol/L |
| serum_creatinine_umol_L | 血肌酐 | 数值 | 男59-104, 女45-84 | 单位：μmol/L |
| egfr_ml_min_1_73m2 | 估算肾小球滤过率 | 数值 | ≥90 | 单位：ml/min/1.73m² |
| uacr_mg_g | 尿白蛋白/肌酐比值 | 数值 | <30 | 单位：mg/g |
| potassium_mmol_L | 血钾 | 数值 | 3.5-5.3 | 单位：mmol/L |
| sodium_mmol_L | 血钠 | 数值 | 137-147 | 单位：mmol/L |

## 高血压分级标准（中国高血压防治指南2024版）

| 分级 | 收缩压(mmHg) | 舒张压(mmHg) |
|:---|:---|:---|
| 正常血压 | <120 | 和 <80 |
| 正常高值 | 120-139 | 和/或 80-89 |
| 1级高血压（轻度） | 140-159 | 和/或 90-99 |
| 2级高血压（中度） | 160-179 | 和/或 100-109 |
| 3级高血压（重度） | ≥180 | 和/或 ≥110 |
| 单纯收缩期高血压 | ≥140 | 和 <90 |

## 糖尿病诊断标准

| 诊断指标 | 静脉血浆葡萄糖值 |
|:---|:---|
| 空腹血糖（FPG） | ≥7.0 mmol/L |
| 餐后2小时血糖（2hPG） | ≥11.1 mmol/L |
| 随机血糖 | ≥11.1 mmol/L（伴典型症状） |
| 糖化血红蛋白（HbA1c） | ≥6.5% |

## 血压控制目标

| 患者类型 | 目标血压 |
|:---|:---|
| 一般高血压患者 | <140/90 mmHg |
| 高危患者（合并糖尿病/肾病/冠心病） | <130/80 mmHg |
| 老年患者（≥65岁） | <150/90 mmHg |

## 血糖控制目标

| 患者类型 | HbA1c目标 |
|:---|:---|
| 一般成人 | <7% |
| 年轻、病程短、无并发症 | <6.5% |
| 老年、有并发症、低血糖风险高 | <8% |
