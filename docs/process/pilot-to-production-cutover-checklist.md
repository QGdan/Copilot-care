# Pilot-to-Production Cutover Checklist v1

## 概述

本文档定义了 CoPilot Care 系统从试点环境切换到生产环境的检查清单。所有项目必须在切换前完成验证。

---

## 1. 基础设施准备

### 1.1 计算资源
- [ ] 生产服务器已配置并可通过公网访问
- [ ] 服务器资源满足最低要求（CPU: 4核, RAM: 8GB, Storage: 100GB SSD）
- [ ] 负载均衡器已配置（如适用）
- [ ] SSL/TLS 证书已安装并验证

### 1.2 网络配置
- [ ] 防火墙规则已配置（仅开放必要端口）
- [ ] 内部服务通信网络已隔离
- [ ] DNS 记录已更新指向生产环境

### 1.3 数据库
- [ ] 生产数据库已创建
- [ ] 数据库备份策略已配置
- [ ] 数据库连接池参数已优化
- [ ] 数据库访问权限已限制

---

## 2. 应用部署

### 2.1 后端服务
- [ ] 后端代码已通过所有测试 (`npm run gate:all`)
- [ ] 环境变量已配置（不含敏感信息硬编码）
- [ ] LLM Provider API 密钥已安全配置
- [ ] 日志级别已设置为生产模式
- [ ] 健康检查端点 (`/health`) 可访问
- [ ] 规则治理端点 (`/governance/rules/catalog`、`/governance/rules/version`) 可访问
- [ ] 互操作端点 (`/interop/fhir/triage-bundle`) 已通过 SMART scope 校验

### 2.2 前端服务
- [ ] 前端构建已通过验证 (`npm run build --workspace=@copilot-care/frontend`)
- [ ] API 基础 URL 已配置为生产地址
- [ ] 静态资源已部署到 CDN（如适用）
- [ ] 前端错误监控已配置

### 2.3 共享类型
- [ ] 共享类型包已同步到前后端
- [ ] 类型定义无冲突

---

## 3. 安全检查

### 3.1 认证授权
- [ ] Consent Token 验证逻辑已启用
- [ ] Token 白名单已配置
- [ ] 会话超时已设置

### 3.2 数据保护
- [ ] 敏感数据传输使用 HTTPS
- [ ] 日志不包含敏感信息（PII）
- [ ] 数据库连接使用加密

### 3.3 访问控制
- [ ] 管理后台访问已限制
- [ ] API 速率限制已配置
- [ ] CORS 策略已正确配置

---

## 4. 功能验证

### 4.1 核心流程
- [ ] 分诊流程端到端测试通过
- [ ] 流式响应正常工作
- [ ] 错误处理返回正确的错误码
- [ ] 幂等性验证通过
- [ ] `ruleGovernance` 字段在成功/失败响应中均可追踪

### 4.2 专家系统
- [ ] 四专家运行时配置正确
- [ ] LLM Provider 连接正常
- [ ] 降级逻辑（无 LLM 时）正常工作

### 4.3 场景回放
- [ ] A-F 基线场景全部通过
- [ ] 对抗性场景全部通过
- [ ] 纵向随访场景全部通过
- [ ] 多站点复制验证通过

---

## 5. 性能验证

### 5.1 响应时间
- [ ] 平均响应时间 < 5秒
- [ ] P95 响应时间 < 10秒
- [ ] 流式首字节时间 < 1秒

### 5.2 吞吐量
- [ ] 并发用户数满足预期
- [ ] 系统资源使用率在合理范围

### 5.3 稳定性
- [ ] 长时间运行无内存泄漏
- [ ] 错误恢复机制正常

---

## 6. 监控与告警

### 6.1 日志
- [ ] 结构化日志已配置
- [ ] 日志聚合系统已接入
- [ ] 日志保留策略已设置

### 6.2 指标
- [ ] 关键指标已采集（延迟、错误率、吞吐量）
- [ ] 治理仪表板数据正常

### 6.3 告警
- [ ] 关键告警规则已配置
- [ ] 告警通知渠道已验证
- [ ] 值班响应流程已确定

---

## 7. 运维准备

### 7.1 文档
- [ ] 运维手册已完成
- [ ] 故障排查指南已完成
- [ ] API 文档已更新

### 7.2 回滚计划
- [ ] 回滚脚本已准备
- [ ] 回滚验证步骤已定义
- [ ] 数据回滚策略已确定

### 7.3 应急响应
- [ ] 应急联系人清单已更新
- [ ] 应急响应流程已演练
- [ ] Stop-loss 触发条件已明确

---

## 8. 合规与审计

### 8.1 审计日志
- [ ] 审计事件完整记录
- [ ] 审计日志不可篡改
- [ ] 审计日志保留期限满足要求
- [ ] 审计日志可关联 `ruleGovernance.evidenceTraceId`

### 8.2 合规检查
- [ ] 数据处理符合隐私法规
- [ ] 医疗安全边界已验证
- [ ] 用户同意机制正常工作

---

## 9. 切换执行

### 9.1 切换前
- [ ] 所有检查项已完成
- [ ] 相关人员已通知
- [ ] 维护窗口已确定

### 9.2 切换中
- [ ] 旧环境流量已停止
- [ ] 新环境已启动
- [ ] 健康检查通过
- [ ] 烟雾测试通过

### 9.3 切换后
- [ ] 流量已切换到新环境
- [ ] 监控指标正常
- [ ] 用户反馈渠道已开放
- [ ] 回滚准备就绪（保留 24 小时）
